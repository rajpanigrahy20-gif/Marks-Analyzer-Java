# Student Marks Analyzer (Java)

A simple Java console application that analyzes a student's marks.

## Features
- Store marks of 5 subjects
- Print all subject marks
- Find highest mark
- Find lowest mark
- Calculate average
- Assign grade based on average

## Technologies Used
- Java

## Concepts Practiced
- Arrays
- For Loop
- If-Else
- Variables
- Arithmetic Operations

## Sample Output

Student Marks:
Subject 1: 85
Subject 2: 90
Subject 3: 78
Subject 4: 95
Subject 5: 88

Average Marks = 87.2
Highest Marks = 95
Lowest Marks = 78
Grade = B
